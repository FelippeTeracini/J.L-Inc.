Insperbot 42
Prévia

Este foi um projeto feito como atividade extraclasse para a entidade estudantil do Insper SMASH. O projeto se trata de um robô de batalha.

O Projeto

Esse projeto foi feito em uma equipe de 4 pessoas. Foi feito um robô do estilo drum com o objetivo de competir na Winter Challenge, uma competição de batalha de robôs. Esse projeto foi feito nas férias e foi muito bom para aprender diversas coisas interessantes.

Resultados

Como resultado final tivemos um robô de batalha feito do 0, e ganhamos uma batalha e perdemos outra na competição.
