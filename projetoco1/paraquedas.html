Salto de Paraquedas
Prévia

Este projeto foi o projeto final de Modelagem e Simulação do Mundo Físico, que tinha o intuito de modelar algum caso de corpo livre. No caso desse projeto, foi escolhido o salto de um paraquedista e tinha como objetivo descobrir para quais altitudes de abertura do paraquedas ele acertaria a área demarcada.

O Projeto

O projeto foi feito em duplas sorteadas aleatóriamente. Foram feitas modelagens do sistema físico, acompanhadas de equações que caracterizam tal sistema, com a finalidade de plotar gráficos para melhor compreender o que acontece durante esta trajetória.

Resultados

Como resultado final tivemos as alturas exatas de abertura de paraquedas para que o paraquedista alcançasse o seu alvo.
