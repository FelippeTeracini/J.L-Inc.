Insper Poker
Prévia

Esse é o projeto final para a materia Design de Software, no qual o objetivo era criar um jogo, aplicativo, site, ou qualquer outra coisa complexa utilizando os nossos conhecimentos de Python que nos foram ensinados ao longo do primeiro semestre de engenharia do Insper.

O Projeto

O projeto se trata de um jogo de Poker, feito para ser jogado contra um computador de inteligência artificial razoavelmente complexa. O jogo foi feito por mim e por outros quatro colegas que compunham nosso grupo. Para que todos aprendessem o máximo possível, separamos o trabalho de forma que todos fizessem um pouco de tudo, como fazer os sprites das cartas, programar a lógica de comparar mãos, passar as funções para o Pygame (módulo do Python utilizado para fazer o jogo), entre vários outras tarefas desafiadoras.

Resultados

O resultado final do projeto agradou todos os membros da equipe, contudo sabemos que poderiamos melhorar o jogo resolvendo bugs, trabalhando melhor na interface, e até tentando fazer com que o jogo fosse jogável através da internet contra outras pessoas.

Anotações Finais

Fazer esse projeto foi muito interessante, pois levou nossos conhecimentos para além da sala de aula, visto que tivemos que aprender coisas que não nos foram ensinadas já que não eram essenciais para o projeto, como por exemplo o Pygame.
